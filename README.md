# 康纳同学 Connor

康纳同学是一款本地优先的 macOS Agent 助手。你把任务交给它，它会在后台调用技能、数据源和本地工具来推进工作，同时记住上下文，让后续协作更连贯。

它不只是聊天，更是一个能帮你做事的 Agent。

---

## 你可以用康纳同学做什么？

康纳同学能处理的任务范围很广。以下是常见的任务类型：

### 写代码、改代码、排查问题

康纳同学可以直接读取和编辑你本机的代码文件，执行构建和测试命令：

- "帮我看看这个 Swift 文件为什么编译不过。"
- "在这个项目的 `Sources/` 下新增一个 `EmailValidator` 类型。"
- "把这个函数里的硬编码改成配置项。"
- "运行测试，把失败的用例修好。"
- "帮我重构这个模块，拆成更小的文件。"

适合搭配工作目录和技能使用。修改代码前会请求确认。

### 编辑文档和笔记

康纳同学可以直接创建、编辑和整理 Markdown 文档：

- "帮我把这份会议记录整理成结构化的 Markdown 文档。"
- "在项目 README 里补充安装步骤和配置说明。"
- "把这几篇笔记合并成一份完整的方案文档。"
- "帮我把这段文字改得更清晰，保留所有技术细节。"
- "根据这个模板帮我生成一份设计文档。"

### 整理和分析资料

把文件、网页、附件交给康纳同学，它会帮你提炼信息：

- "这几份 PDF 的核心结论分别是什么？帮我合并成一份摘要。"
- "帮我分析这份 CSV 数据的趋势和异常。"
- "把这份日志里的错误提取出来，按频率排序。"
- "基于这些附件，帮我写一份决策建议。"

### 处理邮件

康纳同学可以读取邮件、整理内容、起草回复：

- "帮我看看今天有没有需要回复的重要邮件。"
- "根据这封邮件帮我起草回复，不要直接发送。"
- "把最近一周的邮件整理成待办清单。"
- "帮我检查这封邮件的附件是否和项目相关。"

### 安排日程和任务

康纳同学可以查询日程、创建任务、设置自动化：

- "看看我接下来三天的日程，帮我找出适合深度工作的时间段。"
- "把这个项目拆成里程碑和下一步行动。"
- "每天早上 9 点给我发一份今日待办。"
- "当邮件里出现「紧急」字样时，自动提醒我。"

### 浏览网页和做研究

康纳同学可以打开浏览器，帮你搜索、阅读和分析网页内容：

- "帮我搜索这个技术方案的最新进展，整理成对比表格。"
- "打开这个网页，帮我提取关键信息。"
- "帮我比较这两篇博客的观点，找出共识和分歧。"

### 维护人际关系

康纳同学可以记录和管理人物档案、组织关系和联系线索：

- "帮我记录这个人的背景和我们之间的关系。"
- "根据最近的邮件往来，更新这个联系人的信息。"
- "下次联系这个人时，我应该注意什么？"

### 批量处理和自动化

康纳同学可以执行重复性工作，省去手动操作：

- "帮我把这 20 个文件按格式重命名。"
- "检查项目里所有 TODO 注释，整理成清单。"
- "每周自动刷新 RSS 和邮件缓存。"

## 像和同事一样对话

你不需要学习特殊指令。直接用自然语言描述任务，康纳同学会结合当前会话、可用技能、数据源和长期记忆来完成：

```text
帮我把这段资料整理成行动清单。
```

```text
基于这几个附件，帮我总结关键结论。
```

```text
继续推进刚才那个项目计划。
```

```text
帮我检查这段代码为什么失败，修复后跑一下测试。
```

```text
根据这封邮件帮我起草回复，先给我确认。
```

如果你不确定从哪开始，也可以先让康纳同学帮你澄清问题：

```text
我想研究一个产品方向。请先问我 3 个澄清问题，然后帮我形成一个初步计划。
```

## 管理会话和笔记

左侧边栏提供两种常用入口：

- **新建会话**：适合所有类型的任务——问答、研究、写作、开发、整理、自动化。
- **新建或导入笔记**：适合把资料、想法或 Markdown 内容整理成可继续扩展的笔记。笔记也是一种会话，输入体验更偏向长文本和整理沉淀。

---

## 连接你的数据源

康纳同学可以按需连接和读取多种数据源：

- **日历**：查看本机日历、ICS/Webcal、CalDAV 等日历源。
- **邮件**：读取邮件、整理内容、创建草稿，发送前需要你确认。
- **RSS**：订阅和阅读 RSS / Atom / JSON Feed。
- **人际关系**：记录人物档案、联系方式、组织归属和结构化关系。
- **外部工具和服务**：在需要时接入更多工具能力。

建议按需要逐步连接，不需要一开始配置所有能力。

---

## 使用内置浏览器

在会话输入栏可以打开浏览器工作区：

- 一边浏览网页，一边让康纳同学帮你分析。
- 把当前页面或选中文本放入上下文。
- 基于网页内容提问、总结、翻译、比较或继续写作。
- 保留与当前会话绑定的浏览状态。

---

## 使用技能

输入 `/` 或点击技能入口，可以选择当前会话使用的技能。

技能告诉康纳同学"这轮任务应该按某种专业方式完成"，例如研究、写作、开发、排查、总结、规划等。技能不会替代你的指令，而是补充任务风格和处理流程。

## 第一次使用建议

### 第一步：设置模型连接

打开 **设置 → 模型连接**，选择一个模型提供方并填写必要配置。

可选连接包括：

- OpenAI / OpenAI-compatible
- Anthropic
- DeepSeek
- 小米 MiMo
- GitHub Copilot
- 本地模型，例如 Ollama 或本地兼容网关

API Key 会加密保存在本机，不会明文写入设置文件、项目文件或 Git 仓库。

### 第二步：新建一个会话

点击左侧 **新建会话**，直接输入你的任务。可以从一个简单的任务开始，例如：

```text
请帮我整理今天要推进的三件事，并给出下一步行动。
```

### 第三步：设置当前会话工作目录

如果你希望康纳同学处理本地项目文件，可以在输入栏附近设置当前会话的工作目录。设置后，康纳同学会围绕这个目录读取资料或执行你批准的本地操作。

### 第四步：按需要连接数据源

如果你的任务需要日历、邮件、RSS、人际关系或外部工具，可以在左侧 **数据源** 或 **设置** 中配置。

## 权限与确认

康纳同学会尽量把重要操作放在你能理解和确认的流程里：

- **读取本地文件**：通常需要你选择或设置相关工作目录。
- **修改文件或执行命令**：在高风险操作前会让你确认。
- **发送邮件**：康纳同学可以起草邮件，但发送前需要你确认。
- **使用 API Key**：密钥加密保存在本机，不会直接显示在设置文件中。
- **连接数据源**：建议只连接当前任务需要的数据源。

---

## 康纳同学如何"记住"上下文？

康纳同学会在后台整理重要资料和上下文，让后续工作更连续。你不需要直接编辑这些记忆，也不需要管理图谱：

- 它会保留重要证据和上下文。
- 它会把零散信息整理成可复用的工作记忆。
- 它会帮助理解人、项目、概念、资料和关系。
- 重要内容会经过整理和验证，不是直接当作事实。

这套记忆能力的目标，是让康纳同学在长期工作中更可靠、更连续。

---

## 本地优先

康纳同学是 macOS 原生应用，运行数据主要保存在你的本机，包括会话、设置、数据源、技能、任务、附件、搜索索引、后台记忆数据和运行日志。你通常不需要手动编辑这些文件，通过应用界面管理即可。
