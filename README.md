# 康纳同学 Connor

康纳同学是一款本地优先的 macOS Agent 助手。你可以把写作、研究、开发、资料整理和日常事务交给它，并在同一个会话里持续推进。

它不只是回答问题，也可以在你授权后读取资料、操作工作目录、使用数据源和执行工具。重要操作仍由你确认。

## 主要能力

- **对话与长期协作**：围绕同一个任务持续沟通，并保留有用的上下文。
- **文件和代码处理**：阅读、创建和修改本地文件，运行构建、测试与诊断命令。
- **资料整理**：分析文档、网页、附件和笔记，生成摘要、报告或行动清单。
- **内置浏览器**：一边浏览网页，一边提取、比较和整理页面信息。
- **数据源连接**：按需使用邮件、日历、RSS 和人际关系等本地数据。
- **任务与自动化**：管理待办事项，并执行定时或事件触发的工作。
- **技能扩展**：为研究、写作、开发和排查等场景选择专门的工作方式。

## 开始使用

### 1. 安装应用

康纳同学支持 macOS 14 及以上版本。请根据 Mac 的处理器选择安装包：

- Apple M 系列芯片：选择文件名包含 `arm64` 的版本。
- Intel 芯片：选择文件名包含 `x86_64` 的版本。

打开 DMG，将“康纳同学”拖入“Applications”文件夹。

当前公开安装包使用 Ad-hoc 签名，没有经过 Apple 公证。首次启动时：

1. 在“应用程序”文件夹中打开“康纳同学”。如果 macOS 提示 Apple 无法验证，请点击“完成”。
2. 打开“系统设置 → 隐私与安全性”，向下滚动到“安全性”。
3. 在“已阻止使用‘康纳同学’”旁点击“仍要打开”，按系统要求完成身份验证。
4. 警告再次出现时点击“打开”。

完成一次确认后，后续可以正常双击运行，不需要执行终端命令。“仍要打开”仅在尝试启动应用后出现；受组织管理的 Mac 可能不允许用户自行放行。

### 2. 设置模型连接

打开“设置 → 模型连接”，选择模型提供方并填写所需配置。支持的连接包括 OpenAI、Anthropic、DeepSeek、小米 MiMo、GitHub Copilot、Ollama 以及兼容接口。

API Key 会加密保存在本机，不会以明文写入项目文件或会话内容。

### 3. 创建第一个会话

新建会话后，直接描述你希望完成的结果，例如：

```text
帮我整理这份资料，提炼结论并列出下一步行动。
```

```text
检查这个项目为什么构建失败，修复后运行相关测试。
```

```text
查看我接下来三天的安排，找出两个适合集中工作的时间段。
```

涉及本地项目时，请先为当前会话选择工作目录。需要邮件、日历或其他数据时，再按需连接对应数据源。

## 会话、笔记与技能

- **新建会话**适合问答、研究、开发、整理和自动化任务。
- **新建或导入笔记**适合处理 Markdown、长文本和需要持续沉淀的资料。
- 输入 `/` 或使用技能入口，可以为当前任务选择更专门的工作流程。

你不需要学习固定指令。说明目标、限制和期望输出即可；如果需求还不明确，也可以让康纳同学先提出澄清问题。

## 权限与确认

康纳同学只在相关功能需要时请求系统权限，例如联系人、日历、麦克风、语音识别、通知或文件访问。macOS 的权限提示与首次启动确认是两套独立机制。

修改文件、执行高风险命令和发送邮件等操作会遵循应用内的确认与审计流程。建议只授权当前任务需要的目录、数据源和能力。

## 本地数据

会话、设置、任务、附件、数据源缓存、搜索索引和运行日志主要保存在本机。记忆系统在后台整理有来源的上下文，帮助后续会话保持连续；它不是一个需要用户手动维护的图谱编辑器。

模型请求仍会按照你选择的模型提供方发送。是否离开本机取决于所用模型连接和具体任务，请根据自己的数据要求选择服务。
